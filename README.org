* Blorgify
  Build personal blog from org-mode files.

** Usage
   #+begin_src shell
     blorgify [PATTERN.org]
   #+end_src
   This generates ~index.html~ for each ~PATTERN.org~.

   *Example:*
   #+begin_src shell
     blorgify './*/README.org'
   #+end_src
